Bakery Management System

Tech Stack: Python, Text Files
This is a command-line project developed during school days using Python. It helps bakery owners manage their business more easily by automating daily tasks like billing, transactions, stock checking, and more.

System Features:

  • money.txt – to store available funds
  • pass.txt – to store passwords
  • stock.txt – to store items and their quantities

No traditional database is used. Instead, text files and custom logic handle all operations. This project reflects early learning and practical implementation skills.
